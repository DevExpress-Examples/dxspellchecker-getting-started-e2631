Developer Express Example
DXSpellChecker - Getting Started

This tutorial demonstrates how to use the SpellChecker (http://documentation.devexpress.com/#WPF/clsDevExpressXpfSpellCheckerSpellCheckertopic) component to create a simple WPF application with the spell-checker functionality. It can be useful for you in solving this task if you are a first-time user of DXSpellChecker for WPF.

You can find sample updates and versions for different programming languages here:
http://www.devexpress.com/example=E2631.
